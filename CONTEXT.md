# Domain Glossary

## Gemini Auth
The OAuth 2.0 flow initiated by spawning the `gemini login` CLI command. Involves intercepting the Google Accounts authorization URL (`https://accounts.google.com/o/oauth2/`) and monitoring the filesystem for the resulting credentials.

## Credentials File
The JSON file generated by the Gemini CLI upon successful authentication, located at `~/.gemini/oauth_creds.json`, containing the access and refresh tokens.

## Google Accounts File
The JSON file generated by the Gemini CLI upon successful authentication, located at `~/.gemini/google_accounts.json`, containing the currently active email address.

## Gemini Profile
A Code Switcher configuration entity representing the Gemini provider setup. It includes OAuth tokens, active account email, API keys, and model settings. It explicitly omits usage tracking properties due to Google Cloud Platform API limitations.
